# 🎮 Landing Page Clipszoka

Landing page personalizada com tema Twitch para centralizar informações e links do projeto Clipszoka.

## 🎨 Características

- **Design moderno** inspirado na Twitch
- **Tema roxo** (#9146FF) oficial da Twitch
- **Totalmente responsivo** para mobile e desktop
- **Cards expansíveis** com animações suaves
- **Animações de entrada** elaboradas
- **Vídeo integrado** "It's Alanzoka"
- **Links para redes sociais** do Alanzoka

## 📋 Conteúdo

### Cards Implementados:
1. **Alanzoka** - Biografia do streamer com foto e links para Twitch, YouTube e Instagram
2. **Sobre** - História do projeto Clipszoka e apresentação do admin Sérgio
3. **Zokinhas** - Informações sobre o grupo da comunidade no Twitter/X

## ✨ Recursos Especiais

- **Animação de entrada** - Foto de perfil com rotação e borda animada
- **Vídeo principal** - "It's Alanzoka" com controles
- **Cards expansíveis** - Clique para expandir/colapsar conteúdo
- **Favicon personalizado** - Letra "C" em gradiente roxo
- **Ícones sociais** - Links diretos para redes do Alanzoka
- **GIF animado** - No card Zokinhas

## 🚀 Deploy

Projeto configurado para deploy na **Vercel**.

### Como fazer deploy:

1. Instale a Vercel CLI (opcional):
```bash
npm i -g vercel
```

2. Faça login na Vercel:
```bash
vercel login
```

3. Deploy:
```bash
vercel
```

Ou simplesmente arraste a pasta no painel da Vercel: [vercel.com](https://vercel.com)
